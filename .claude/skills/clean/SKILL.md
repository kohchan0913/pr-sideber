# コードクリーンアップ

`refactor-cleaner` agent を起動して $ARGUMENTS の範囲 (未指定ならプロジェクト全体) を清掃する。

## 検出対象

- 未使用の export / 関数 / 変数 / 型 / pnpm パッケージ / crate
- コメントアウトされたコードブロック
- 50行超の関数、3段超のネスト、マジックナンバー

## 制約

- 機能追加はしない。削除と簡素化のみ
- RISKY な変更はユーザーに確認を取る
- 除去後にテスト実行で壊れていないことを確認する
